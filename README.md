# Transfer

Upload to <https://transfer.sh>

## Install

With [fisherman]

```
fisher transfer
```

## Usage

```fish
transfer my-file.txt
```

```fish
transfer my-file.txt my-file-new-name.txt
```

```fish
echo my message text | transfer my-message.txt
```
